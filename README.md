# Iori Birthday
미나세 이오리 생일축하 페이지

## 설치
1. clone repository
2. 실행 'npm install'

        npm install

## 빌드
1. 개발 빌드

        gulp

2. 프로덕션 빌드

        gulp clean
        npm run-script build
